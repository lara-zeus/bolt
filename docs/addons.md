## Addons

Sky comes with some addons, currently we have:
- FAQ

## FAQ Addons
to set the url of the FAQ from the `zeus-sky.php` config file:
```php
'faq_uri_prefix' => 'faq',
```

to disable any resource, remove it from the config:
```php
'enabled_resources' => [ ... ],
```
